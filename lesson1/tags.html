HTML - язык разметки гипертекста. HTML отвечает за то, какие элементы будут расположены на странице, за то, как они должны выглядеть отвечает CSS.

На логическом уровне сайт представляет собой совокупность веб-страниц (HTML-документов), объединенных между собой единым дизайном, стилем и ссылками.

Каждая веб-страница (HTML-документ) - совокупность HTML-элементов (тегов) и текста.

HTML-теги — основа языка HTML. Они используются для определения элементов в разметке страницы.

Синтаксис тегов:

  • парные теги состоят из двух частей: открывающий тег и закрывающий тег. В них можно вкладывать другие теги или текст

    <имя_тега имя-атрибута="значение_атрибута" имя-атрибута-2="значение_атрибута">содержимое тега</имя_тега>

  • одиночные теги не имеют закрывающего тега, в них нельзя вкладывать другие теги и текст

    <имя_тега имя-атрибута="значение_атрибута"/>
Дополнительные правила::
  • содержимое тега: другой тег или текст;
  • атрибуты тегов содержат дополнительную информацию о теге;
  • атрибуты тегов разделяются между собой пробелом;
  • атрибуты тегов могут быть обязательными и нет;
  • значения атрибутов необходимо писать в двойных кавычках;
  • теги и атрибуты не чувствительны к регистру;
  • если тег или атрибут написан с ошибкой, браузер игнорирует их и отображает, как текст.

Поведение элементов на странице (без использования css стилей)

Разные элементы ведут себя на странице по разному: одни выстраиваются друг за другом, третьи друг под другом, занимая при этом разное простроанство. В зависимости от этого элементы можно разделить на:

Блочные элементы

  1. Занимают всю доступную ширину страницы (если она не установлена)
  2. Если один элемент помещен в другой, то ширина вложенного начинает вычисляться отосительно родителя
  3. Высота (если не установлена) определяется содержимым
  4. Всегда начинаются с новой строки
  5. В блочный элемент можно вкладывать другие блочные и строчные элементы

Строчные элементы:

  1. Являются частью строки
  2. Ширина равна содержимому
  3. Свойства, связанные с размерами не применяются
  4. Несколько строчных элементов переносятся на новую строку при необходимости
  5. Можно выравнивать по вертикали
  6. В строчные элементы можно вкладывать другие строчные элементы и текст, но не блочные элементы

Осовные html теги

<!DOCTYPE html> объявляет тип документа для браузера;

Теги верхнего уровня

  • <html></html> корневой элемент HTML-документа. Является контейнером для всех остальных html-элементов
  • <head></head> элемент-контейнер для метаданных HTML-документа. В него могут быть включены: заголовок, мета теги, подключеие стилей и скриптов. Данная информация не отображается пользователю (кроме title)
  • <body></body> тело документа. Контейнер для элементов, которые отображаются на странице

Основыные теги, вкладываемые в тег head

  1. <title></title> отображается как заголовок вкладки в браузере
  2. <meta charset="UTF-8"> указание кодировки документа
  3. <meta name="description" content="описание"> поисковики используют для индексации
  4. <meta name="keywords" content="one, two, etc"> список ключевых слов (касающиеся конкретной страницы)
  5. <meta name="author" content="описание"> указание автора
  6. <style></style> добавление стилей
  7. <link href="путь_к_файлу" rel="stylesheet"> подключение файла со стилями
  8. <script src="путь_к_файлу"></script> добавление js скриптов

Основыные теги, вкладываемые в тег body

Семантические теги HTML5 (являются блочными элементами)

  1. <header></header> контейнер для вводной информации сайта. Может содержать логотип, меню сайта и тп
  2. <main></main> контейнер для основного содержимого страницы. На одной странице должен быть только один тег main
  3. <aside></aside> контейнер для дополнительного содержимого страницы (страница не потеряет своей сути, если данный контент не будет отображен)
  4. <footer></footer> для обозначения подвала сайта
  5. <nav></nav> контейнер для размещения навигации (меню) сайта
  6. <section></section> определяет раздел страницы, обычно с заголовком
  7. <article></article> определяет независимую часть содержимого страницы. В одном section может быть несколько article
  8. <address></address> Задает контактные данные автора/владельца документа или статьи.

Основыные теги для отображения текста

  1. <h1></h1> / <h2></h2> / <h3></h3> / <h4></h4> / <h5></h5> /<h6></h6> - теги заголовка с 1го по 6й уровень (блочные элементы)
  2. <р></р> - текстовый абзац (блочный элемент)
  3. <blockquote></blockquote> - для выделения цитат (блочный элемент)
  4. <q></q> - для выделения коротких цитат (строчный элемент)

Теги для отображения списков

  1. <ul></ul> - маркированный список (блочный элемент)
  2. <ol></ol> - нумерованный список (блочный элемент)
  3. <li></li> - элемент нумерованного или маркированного списка (блочный элемент)

Тег для добавления ссылок на страницу

  1. <a href="ссылка"></a> - ссылки могут быть абсолютные (https://github.com) и относительные (contacts.html) (строчный элемент)
  2. <a href="ссылка" download ></a> - download - ссылка для скачиваний
  3. <a href="ссылка" target="_self" ></a> - target="_self" - открывает документ в текущей вкладке
  4. <a href="ссылка" target="_parent" ></a> - target="_parent" - открывает документ в родительской вкладке или ведет себя как self
  5. <a href="ссылка" target="_top" ></a> - target="_top" - открывает документ в окне верхнего уровня или ведет себя как self

Тег для добавления изображений на страницу

  • <img src="путь_к_изображению" alt="текст_если_картинка_не_загрузится"> (строчный элемент)

Тег - контейнер для группировки элементов

  • <div></div> (блочный элемент)

Теги для формирования таблиц

  1. <table></table> - границы таблицы
  2. <caption></caption> - заголовок таблицы
  3. <tr></tr> - строка таблицы
  4. <th></th> - ячейка таблица (отображается жирным)
  5. <td></td> - ячейка таблица
